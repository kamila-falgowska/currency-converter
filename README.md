## Description

This simple currency converter is designed to help you convert amounts from three different currencies to Polish złoty (PLN). You can convert Polish Zloty to:
- Euro (EUR)
- Dollar (USD)
- Norwegian Krone (NOK)

## Technology used
- HTML
- CSS
- Java Script
- BEM convention
- Normalize CSS
